Apa Itu UPC ?

UPC (UAJY Programming Competition) merupakan bagian dari event IFEST UAJY. Acara ini telah diselenggarakan sebelumnya pada tahun 2015 pada IFEST #3 untuk tingkat SMA/SMK/Sederajat Se-Jawa dan Bali. Acara ini direspon sangat baik oleh siswa-siswi tingkat SMA/SMK, maka kami memutuskan untuk kembali mengadakan UPC lagi pada IFEST #4 untuk tingkat SMA/SMK/Sederajat. Kegiatan ini diselenggarakan untuk mengembangkan kemampuan siswa-siswi SMA/SMK dan berharap dengan adanya kompetisi ini siswa-siswi menjadi tertarik dalam bidang teknologi informasi. UPC pada IFEST #4 ini bertemakan Code Rush. Peserta akan beradu kemampuan codingnya pada bahasa pemrograman C, C++, atau Pascal.
